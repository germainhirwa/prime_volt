# Prime Volt — Electric Vehicle Sales & Distribution Platform

Premium EV sales platform for the Rwandan market. Built with HTML, CSS, and GSAP for high performance and premium aesthetics.

## Tech Stack
- Vanilla HTML5/CSS3
- Modern JavaScript
- GSAP (GreenSock Animation Platform) for cinematic animations
- Responsive, mobile-first design

## Project Structure
- `index.html`: Core layout and SEO
- `css/styles.css`: Custom design system
- `js/`: Interactive logic, filtering, and animations
